Personal-Porfolio
=================

Teng's Porfolio


这份个人档案做为技术展示适用于*现代浏览器（Chrome， Firefox， IE10等）*，大多用CSS3的新特性（transition， transform等），为达到最佳显示效果，并未做降级兼容性处理。

采用了最新的bootstrap v3框架，为流式布局，随页面大小变动而有不同的表现方式。导航栏、技术框部分均用icon font取代了传统图片，可随时根据具体情况改变颜色。并用了transition做平滑显示效果，再采用transform做一个小动画的效果。

script.js简单的做了一个页面平滑滚动以及回到顶部的jQuery特效。
